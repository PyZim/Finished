Running

1. Checkout wafer to ../wafer
2. (optional) Create a localsettings.py containing local settings.
3. npm install
4. $(npm bin) bower install
5. bower install
6. ./manage.py syncdb
7. ./manage.py migrate
8. ./manage.py runserver
